# AntiGrief

Плагин защиты для Minecraft серверов на Folia/Paper 1.21+. Делает то, чего нет в других плагинах — привязывает аккаунты игроков через соцсети, ловит мультиаккаунты по отпечаткам браузера и устройства, и не даёт забаненным вернуться просто сменив браузер.

## Что умеет

**Авторизация через соцсети** — игроки заходят на сайт mc.твой-домен.ru и логинятся через Discord, Google или Telegram. Ник в Minecraft привязывается к веб-аккаунту, и без подтверждения через соцсеть никто другой под этим ником не зайдёт.

**Защита от кражи ника** — если кто-то заходит с нового IP под чужим ником, его замораживает. Разморозка только через OAuth2 на сайте — злоумышленник не знает пароль от соцсети владельца.

**VPN/прокси детекция** — proxycheck.io v3 + ip-api.com как фоллбэк. Кикает за VPN, прокси, Tor и хостинг. Белый список обходит эту проверку.

**Гео-фильтр** — пускать только из СНГ (настраивается). Остальных кикает.

**Отпечатки браузера (2 уровня)**:
- Уровень 1 — FingerprintJS v5, уникальный visitorId для каждого браузера
- Уровень 2 — deviceHash, стабильный при смене браузера на одном устройстве (GPU + шрифты + CPU + RAM + экран + timezone)

Если забанить игрока — банится его аккаунт, все его браузерные отпечатки, и все его устройственные отпечатки. Смена браузера не поможет — deviceHash совпадёт.

**WebRTC leak** — пытается получить реальный IP мимо VPN через STUN-сервер.

**Discord аудит** — все действия админов и автоматические события логируются в Discord канал через SOCKS5 прокси.

**Бан-система**:
- `/angri ban <ник> [причина] [1h30m/7d/1y...]` — бан по нику (все IP + аккаунт + отпечатки)
- `/angri unban <ник>` — полный разбан
- `/angri banlist` — список забаненных с причинами и сроками
- Поддержка комбинированного времени: `30m`, `1h30m`, `2w3d12h`, `1M`, `1y`

## Команды

| Команда | Описание |
|---------|----------|
| `/angri` | Ссылка на сайт для привязки аккаунта |
| `/angri status` | Статистика плагина |
| `/angri reload` | Перезагрузка конфига |
| `/angri wl add/remove/list` | Белый список (ники/IP) |
| `/angri ban <ник/IP> [причина] [время]` | Забанить |
| `/angri unban <ник/IP>` | Разбанить |
| `/angri banlist` | Список банов |

## Установка

1. Положить AntiGrief-2.0.6.jar в папку plugins
2. Запустить сервер — создастся конфиг и messages_ru.yml
3. Настроить config.yml (OAuth2 ключи, домен, Discord токен)
4. Настроить nginx как обратный прокси на порт 8199
5. Перезапустить сервер

## Требования

- Folia или Paper 1.21+
- Java 21+
- nginx для HTTPS (Let's Encrypt)
- Домен для сайта
- OAuth2 ключи: Discord, Google, Telegram
- (Опционально) SOCKS5 прокси для Discord бота и VPN API из РФ
- (Опционально) proxycheck.io API ключ (без ключа 100 запросов/день)

## Как это работает

1. Игрок заходит на сервер → PreLogin проверяет баны, VPN, страну
2. Если ник не привязан → заморозка, просит зайти на сайт
3. Игрок логинится через соцсеть → ник привязывается к аккаунту
4. При загрузке профиля сайт собирает отпечатки браузера и устройства
5. Если отпечаток совпадает с забаненным → автоматический бан
6. Если устройство совпадает с забаненным → автоматический бан
